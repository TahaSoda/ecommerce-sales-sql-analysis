# E-Commerce Sales Analysis (SQL Project)

## About This Project

This project is a small data analysis case study where I explored an **e-commerce sales dataset** using SQL and visualized the results in Excel.

The goal was to practice real-world data analyst tasks such as calculating business KPIs, analyzing sales trends, identifying top-performing products, and understanding customer purchasing behavior.

The final results were summarized in an **Excel dashboard** to make the insights easier to understand.

---

## Dataset

File: `ecommerce data.csv`

The dataset contains 10k** e-commerce orders** with information about customers, products, sales, and profit.

Main columns in the dataset include:

* **OrderID** – unique identifier for each order
* **CustomerID** – customer identifier
* **OrderDate** – date of the purchase
* **Region** – geographical sales region
* **Category** – product category
* **Product** – product name
* **Quantity** – number of items purchased
* **UnitPrice** – price per item
* **Sales** – total revenue for the order
* **Profit** – profit generated from the order
